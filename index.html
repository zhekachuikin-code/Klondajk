/ Генератор описань
Gemini · УКР + РУС · HTML · Авто · Пакет
📖 Як користуватися
1

Вставте посилання

Повний URL зі сторінки товару на будь-якому маркетплейсі.

2

Додайте примітки

Бренд, акції, доставка, гарантія — AI врахує їх в описі.

3

⚡ Авто — одна кнопка

Запускає весь цикл: опис → SEO → назва → таблиця автоматично (~3 хв).

4

Або покроково

«Створити» — HTML з брендом. «БЕЗ БРЕНДА» — без згадки бренду. Потім SEO → Назва → Таблиця.

5

📦 Список посилань

Вставте кілька URL (кожне з нового рядка) — обробка пройде по черзі.

6

📁 Файл XLS/CSV

Завантажте прайс із колонками: Артикул, Найменування, РРЦ, Опис.

💡 Поради

Товар має бути публічно доступний (без авторизації).

РУС-версія генерується як переклад УКР-версії.

SEO-ключі — лише інформаційні, без комерційних слів.

В пакетному режимі примітки застосовуються до всіх товарів.
